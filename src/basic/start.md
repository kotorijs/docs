# 快速开始

## 安装 Node.js

前往 [Node.js 官网](https://nodejs.org) 根据系统版本选择对应的安装方式，确保 Node.js 版本 >= 17.9.0。输入以下命令确认安装成功：

```bash
node -v
npm -v
```
